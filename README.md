В этом репозитории будут храниться домашние задания по курсу автоматизации тестирования на Python.

## Клонирование проекта

```bash
git clone <git@github.com:ElenaKurilo/ElenaKruhliak-QAP.git>
cd ElenaKruhliak-QAP
```

## Правила именования веток

Для каждого домашнего задания нужно создавать отдельную ветку.

Формат названия ветки:

```bash
homework_<название_урока>
```

Пример для урока по Git:

```bash
homework_git
```
